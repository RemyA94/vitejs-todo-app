# TODO VITE

pasos para ejecutar el proyecto: 

1. Clonar el repositorio
2. Ejecutar ```npm install``` para rescontruir los modulos de node
3. Correr el devServer ```npm run dev```
4. Abrir proyecto en: ```http://localhost:5173/```
